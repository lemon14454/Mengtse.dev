## MERN Shop

一個全端的電商網站，MERN 分別代表 MongoDB、Express、React、NodeJS，而後端部分則照 MVC
架構開發。起初原型來自 Brad Traversy 的 Udemy 課程，原課程則是使用 React Boostrap
及舊版的 Redux 作為開發，不得不說當初跟著做完的確學習了不少東西。但之後我陸續學習了
Webpack、Tailwind、Typescript、Redux Tool Kit 等技術，就將原本的專案打造成全新的樣貌。

## 使用技術

- React
- Tailwind
- Webpack
- Redux Tool Kit
- PaypalAPI
- MongoDB、Express
- Typescript

對我而言其實一直不想自行去開發後端，所以這算是少數有自己著手的全端作品，基本的 MVC 架構及 Middleware
等等還是瞭解的。

## Redux 與 Typesciprt

當初在學習完 Typesciprt 之後想馬上用在作品上面，但我自己覺得要讓舊版 Redux Type Safe
真的是一件麻複雜的事，網路上大家的架構也都不大一樣，後來得知有新出的 RTK 之後似乎就減少了這個難度，
官方本身也很推薦用 TS 來寫心的 RTK。

新舊版比較下來雖然新的 RTK 節省了超級多的程式碼，但舊版撇除 Type Safe 的問題的話還是比較直觀一點。
以目前的經驗來看，我可能還是會選擇用原生的 React Context 來完成 state management。
